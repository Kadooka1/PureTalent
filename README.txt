Plataforma de Vagas Temporárias

📌 Sobre o Projeto

Este projeto tem como objetivo o desenvolvimento de uma plataforma digital voltada para a divulgação e busca de vagas temporárias. A proposta é facilitar a conexão entre empresas que precisam de mão de obra temporária e candidatos em busca de oportunidades rápidas e acessíveis.

O sistema foi pensado para ser simples, intuitivo e eficiente, permitindo que os usuários encontrem vagas de acordo com suas preferências, como cargo, localização ou tipo de trabalho.

🎯 Objetivo

Criar uma plataforma funcional onde usuários possam:

Buscar vagas temporárias

Filtrar oportunidades por cidade ou modalidade (presencial/remoto)

Visualizar detalhes das vagas

Se candidatar de forma prática

🛠️ Tecnologias Utilizadas

Durante o desenvolvimento do projeto, foram utilizadas as seguintes ferramentas:

Figma → criação do protótipo e design das telas

Canva → elaboração da apresentação do projeto

HTML, CSS e JavaScript → desenvolvimento da interface (front-end)

🧩 Funcionalidades

A plataforma conta com as seguintes funcionalidades principais:

Barra de busca para encontrar vagas por palavra-chave ou cargo

Filtros por localização e tipo de trabalho

Categorias populares (ex: TI, Marketing, etc.)

Listagem de vagas com informações detalhadas

Botão para candidatura nas vagas

📌 Considerações Finais

O projeto de vagas temporárias busca resolver um problema comum no mercado de trabalho: a dificuldade de encontrar oportunidades rápidas e acessíveis. Através de uma interface simples e funcionalidades eficientes, a plataforma oferece uma solução prática tanto para candidatos quanto para empresas.
